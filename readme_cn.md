# Snail Image Confuse

Snail Image Confuse 是个简易的图片混淆工具，
用于绕过图片审查，比如在即时聊天软件中。

生成的结果可能有点奇怪，不过实用型可能不错？

目前只支持生成PNG文件，因为我懒得写一些逻辑……
